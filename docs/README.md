# Story CLI Documentation

**Story CLI** is an AI-powered visual novel story builder that helps authors develop rich, consistent narratives through structured wizards and iterative planning workflows, powered by local LLMs.

|-------|-------------|

| [Story Management](./story-guide.md) | Create and manage story projects |
| [Character Management](./character-guide.md) | Create characters with the AI-powered wizard |

## Prerequisites

Before using Story CLI, you'll need:

- **Python 3.11+** - [Download Python](https://www.python.org/downloads/)
- **pip** - Included with Python

See the [Installation Guide](./installation.md) to get started.

## Getting Started

Follow this learning path for the best experience:

1. **[Install Story CLI](./installation.md)** - Get the tool running on your system
2. **[Set Up Ollama](./llm-setup.md)** - Enable AI-powered features (optional but recommended)
3. **[Create Your First Project](./story-guide.md)** - Initialize a story project
4. **[Create Characters](./character-guide.md)** - Use the wizard to build your cast

## Features at a Glance

- **Project Management**: Create and organize visual novel projects
- **Character Wizard**: 5-phase guided character creation
- **AI Assistance**: Name suggestions, appearance expansion, trait generation, backstory development
- **Local-First**: All data stored locally, LLM runs via Ollama (no cloud required)
- **JSON Export**: Clean data format for integration with other tools
